Preface

This describes the use of Java Mission Control.

About the Document

This document contains the following chapters:

  • Chapter 1, "Introduction to Java Mission Control", which contains introductory information about Java Mission Control.

  • Chapter 2, "Using the Java Mission Control Client", which describes the features in this version of Java Mission Control.

  • Chapter 3, "Managing JVM Connections", which describes the communication protocols used by Java Mission Control and how they differ based upon the J2SE version used.

  • Chapter 4, "Accessibility Features in Java Mission Control", provides accessibility features in this version of Java Mission Control.

  • Chapter 6, "Frequently Asked Questions", provides frequently asked questions and answers about Java Mission Control.
